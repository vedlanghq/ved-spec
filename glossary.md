# Lexum Glossary

## Domain

An isolated unit of state and execution.

## State

Persistent data representing system condition.

## Goal

A predicate defining desired system state.

## Transition

A deterministic state transformation step.

## Slice

A bounded unit of execution.

## Effect

An external interaction performed by the system.

## Scheduler

Component responsible for selecting execution steps.

## Determinism

Property that identical inputs produce identical execution outcomes.

## Convergence

Process of moving system state toward goal satisfaction.
