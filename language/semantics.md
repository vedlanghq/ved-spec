# Language Semantics

Ved programs describe evolving system state.

Execution semantics are defined in terms of:

- state space
- goal predicates
- transition functions
- effect intents

## State Space

State is modeled as a structured persistent data graph.

## Goals

Goals define stable conditions over state.

## Transitions

Transitions are deterministic transformations of state.

## Effects

Effects represent interactions with external systems.

Semantics prioritize:

- predictability
- composability
- observability
